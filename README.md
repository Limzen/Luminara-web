# Luminara Frontend

## Project Structure

- `src/components/` - Reusable UI components
- `src/pages/` - Page-level components
- `src/data/` - Dummy data for development
- `src/routes/` - Routing logic
- `src/styles/` - Global and module CSS
- `src/assets/` - Images and static assets

## Development

- Uses React, Vite, and React Router DOM
- Homepage is built based on Figma design
- Backend is under development; dummy data is used for now

## Getting Started

1. `npm install`
2. `npm run dev`
